 <p align="center"> 
  Android Tool in Java for social distancing. Geofencing, Google Maps API, and user
location. alerts when leaving the allowed zone.
  </p>

  

### Features  
 - Google Maps API.
 - Geofencing.
 - User Location alerts.
 - Notification.
 - offline capibale app.
 - Simplified UI.
 
<br>

### Technologies 
- Google Maps.
- Animations.
- Multiple Language support.
- Motion-Layout
